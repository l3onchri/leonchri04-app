# Direttive

Questa directory contiene le SOP (Standard Operating Procedures) in formato Markdown.

## Struttura di una Direttiva

Ogni direttiva dovrebbe definire:
- **Obiettivo**: Cosa si vuole ottenere
- **Input**: Dati o risorse necessarie
- **Tool/Script**: Script in `execution/` da utilizzare
- **Output**: Risultato atteso
- **Casi Limite**: Gestione di errori e situazioni speciali

## Esempio

```markdown
# Nome Direttiva

## Obiettivo
Descrivere cosa si vuole ottenere.

## Input
- Parametro 1
- Parametro 2

## Script da Usare
`execution/nome_script.py`

## Output
Descrivere l'output atteso.

## Casi Limite
- Gestione errore X
- Gestione caso Y
```
